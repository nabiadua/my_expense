# 💸 Expense Tracker with AI Liveness Verification

A professional-grade Flutter mobile application designed for secure personal finance management. This project integrates real-time facial liveness detection to ensure that only the authenticated user can access their financial data.

## 🚀 Features

- **AI Liveness Check**: Biometric verification (blinking and smiling challenges) before accessing the dashboard.
- **Dynamic Balance Engine**: Real-time calculation of Total Balance using the formula: `Income - Expenses`.
- **Offline First**: Powered by **Hive (NoSQL)** for lightning-fast, persistent storage that works without internet.
- **Figma-Inspired UI**: A clean, modern Fintech interface with a focus on visual hierarchy and user experience.
- **Swipe-to-Delete**: Intuitive gestures to manage and remove transaction history.

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev) (Dart)
- **Database**: [Hive](https://pub.dev/packages/hive) (Local Persistence)
- **Biometrics**: `facial_liveness_verification` plugin
- **State Management**: `ValueListenableBuilder` (Reactive UI)
